
# 🔍 Audit Risk Detection System

An AI-powered audit analytics tool that identifies high-risk financial transactions using machine learning.

## 🚀 Features
- Risk scoring using XGBoost
- Interactive Streamlit UI
- Bulk transaction analysis
- Power BI dashboard integration

## 🧠 Problem Statement
Auditors cannot manually review all transactions. This system prioritizes high-risk cases for efficient auditing.

## ⚙️ Tech Stack
- Python
- Scikit-learn / XGBoost
- Streamlit

## 📊 Output
- Risk Score (0–100%)
- Risk Category (Low / Medium / High)
- Audit Recommendations



## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
